# ⚡ Quick Start FinWa untuk UMKM
## Panduan Cepat Mencatat Keuangan via WhatsApp

---

## 📱 Mulai Cepat (30 Detik)

### 💰 Catat Pemasukan
```
dapat orderan 500rb
terima transfer 1jt
gaji masuk 5jt
```

### 💸 Catat Pengeluaran
```
beli bahan 750rb
bayar listrik 500rb
bensin 50rb
```

### 📊 Cek Ringkasan
```
ringkasan bulan ini
cek saldo
lihat transaksi
```

---

## 🎯 Format Angka

| Tulis | Artinya |
|-------|---------|
| `25rb` | Rp 25.000 |
| `50k` | Rp 50.000 |
| `1.5jt` | Rp 1.500.000 |

---

## 📝 Catat Banyak Sekaligus

```
Biaya tanggal 13 Des 2025
1. Bahan baku 500rb
2. Bensin 50rb
3. Makan 25rb
4. Parkir 10rb
```

---

## 💳 Kelola Dompet

```
tambah dompet BCA           → Buat dompet baru
lihat dompet                → Lihat semua saldo
dapet tf 1jt ke BCA         → Catat ke dompet tertentu
```

---

## ✏️ Edit & Hapus

```
edit jadi 100rb             → Ubah nominal
ganti kategori makan        → Ubah kategori
hapus transaksi             → Hapus terakhir
```

---

## ⏰ Pengingat Tagihan

```
ingatkan bayar listrik tgl 20 500rb
lihat pengingat
hapus pengingat 1
```
