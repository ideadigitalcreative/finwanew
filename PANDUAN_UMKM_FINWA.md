# 📱 Panduan FinWa untuk UMKM
## Mencatat Operasional Harian dengan WhatsApp

---

## 🎯 Apa itu FinWa?

**FinWa** adalah asisten keuangan pintar yang membantu Anda mencatat dan menganalisis keuangan bisnis melalui WhatsApp. Cukup kirim pesan seperti ngobrol biasa, tanpa perlu aplikasi tambahan!

---

## 🚀 Cara Memulai

### Langkah 1: Daftar Akun
1. Kunjungi website FinWa
2. Daftar dengan email dan nomor WhatsApp
3. Hubungkan WhatsApp Anda dengan mendaftarkan nomor yang ingin digunakan

### Langkah 2: Siap Mencatat!
Setelah terhubung, langsung kirim pesan ke nomor FinWa untuk mulai mencatat.

---

## 💰 Mencatat Pemasukan (Uang Masuk)

### Format Dasar
Cukup ketik pesan seperti ngobrol biasa:

| Contoh Chat | Penjelasan |
|-------------|------------|
| `dapat orderan 500rb` | Pemasukan dari orderan |
| `terima transfer 1.5jt` | Transfer masuk |
| `dapet bayaran 2jt` | Pembayaran dari customer |
| `gaji karyawan masuk 5jt` | Gaji atau honor |
| `bonus project 750rb` | Bonus atau insentif |
| `customer bayar 350k` | Pembayaran customer |

### Format Angka yang Didukung
- `25rb` atau `25ribu` = Rp 25.000
- `50k` = Rp 50.000
- `1.5jt` atau `1.5juta` = Rp 1.500.000
- `25.000` = Rp 25.000

### Contoh Lengkap UMKM

**Toko/Warung:**
```
dapat jual hari ini 1.2jt
terima bayaran customer 500rb ke BCA
dapet orderan gofood 150rb
```

**Jasa/Freelance:**
```
terima transfer project 3jt
dapat fee design 500rb
honor event 750rb
```

**Kuliner:**
```
hasil jualan 850rb
dapat pesanan katering 2.5jt
omset warung 1.5jt
```

## 📝 Mencatat Banyak Transaksi Sekaligus (BATCH)

### Format untuk Pengeluaran Harian

Saat punya banyak pengeluaran dalam sehari, kirim dalam format list:

```
Biaya tanggal 13 Desember 2025
1. Bahan baku 500rb
2. Bensin motor 50rb
3. Parkir 10rb
4. Makan siang 25rb
5. Ongkir JNE 15rb
```

### Format untuk Pemasukan dari Banyak Customer

```
Uang masuk atas nama
Pak Ahmad 500rb
Bu Siti 350rb
Toko Makmur 1.2jt
Customer Online 275rb
```

### Tips Batch:
- ✅ Sertakan tanggal di baris pertama
- ✅ Gunakan nomor urut untuk setiap item
- ✅ Sertakan nominal di setiap item

## 📸 Mencatat dari Struk/Nota

### Cara Pakai
1. Foto struk belanja/nota dengan jelas

3. FinWa akan otomatis membaca dan mencatat!

- ✅ Pastikan struk tidak kusut
- ✅ Cahaya cukup terang
- ✅ Total belanja terlihat jelas
- ✅ Foto lurus (tidak miring)

## 💡 Tips & Trik untuk UMKM

### 1. Catat Langsung
Jangan tunda! Setiap ada transaksi, langsung kirim pesan ke FinWa.

### 2. Gunakan Format Singkat
- `bahan 500rb` lebih cepat dari `beli bahan baku seharga lima ratus ribu rupiah`

### 3. Pisahkan Dompet
- Buat dompet terpisah untuk kas toko dan pribadi
- `tambah dompet Kas Toko`
- `tambah dompet Pribadi`

### 4. Rutin Cek Ringkasan
- Cek harian: `ringkasan hari ini`
- Cek mingguan: `cashflow minggu ini`
- Cek bulanan: `ringkasan bulan ini`

### 5. Manfaatkan Pengingat
Jangan sampai telat bayar tagihan:
```
ingatkan bayar listrik tgl 15
ingatkan bayar sewa tgl 1 2.5jt
ingatkan bayar wifi tgl 5
```

### 6. Foto Struk Besar
Untuk belanja besar seperti kulakan, foto struk langsung kirim!

## ❓ FAQ (Pertanyaan Umum)

### Salah catat nominal, bagaimana?
```
edit jadi [nominal benar]
```
Contoh: `edit jadi 500rb`

### Lupa catat kemarin, bagaimana?
Tambahkan tanggal di pesan:
```
beli bahan kemarin 500rb
```

### Bisa untuk grup WA?
Saat ini FinWa bekerja via chat pribadi untuk keamanan data Anda.

### Data aman?
Ya, data Anda terenkripsi dan hanya bisa diakses oleh Anda.

### Bisa banyak bisnis?
Gunakan dompet terpisah untuk masing-masing bisnis:
```
tambah dompet Toko A
tambah dompet Toko B
```
